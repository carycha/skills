# 分類搜尋快捷方式

在透過 `google_web_search` 或 `web_fetch` 執行搜尋時，請參考以下關鍵字。

## 開發工具
- **程式碼檢查 (Linting)**: `eslint`, `ruff`, `textlint`, `markdownlint`
- **程式碼格式化 (Formatting)**: `prettier`, `black`, `gofmt`
- **測試 (Testing)**: `jest`, `pytest`, `go test`
- **提交前檢查 (Pre-commit)**: `husky`, `lint-staged`, `pre-commit`

## AI/LLM 整合
- **SDKs**: LangChain, LlamaIndex, LiteLLM
- **提示詞管理 (Prompt management)**: Promptfoo, LangSmith
- **文件處理**: `unstructured`, `pdfplumber`, `mammoth`

## 數據與 API
- **HTTP 客戶端**: `httpx` (Python), `ky`/`got` (Node)
- **驗證 (Validation)**: `zod` (TS), `pydantic` (Python)
- **資料庫 (Database)**: Prisma, SQLAlchemy, Drizzle

## 內容與出版
- **Markdown 處理**: `remark`, `unified`, `markdown-it`
- **圖片優化**: `sharp`, `imagemin`
